Students Mathematics Performance

In this project I used both classification and regression algorithms to predict the grades achieved by secondary education students in their final mathematics exam. The dataset and a detailed description of the 33 variables (both categorical and numerical) can be found in the UCI Machine Learning Repository at the following link: archive.ics.uci.edu/ml/datasets/Student+Performance. The data were collected from 395 Portuguese students attending two different schools.
